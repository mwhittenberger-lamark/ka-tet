# Ka-Tet RPG Website Page Inventory

This file lists the recommended pages, routes, page purpose, audience, content blocks, and build notes.

## Audience Values

- **Public**: Safe for anyone, including prospective players.
- **Player**: Safe for active players; may include light campaign premise but no major spoilers.
- **GM**: Contains spoilers, secrets, hidden mechanics, and campaign truths.

## Spoiler Values

- **None**: Rules, basic lore, public pitch.
- **Light**: Campaign premise, NPC role, non-secret tensions.
- **Major**: Deaths, betrayals, endgame truths, hidden identities.

---

## `/`

### Page: Homepage

**Audience:** Public  
**Spoiler Level:** None  
**Purpose:** Sell the game quickly and direct users to the right starting point.

### Required Content Blocks

1. Hero
   - Title: Ka-Tet: The Path of the Beam
   - Subtitle: A fan-made tabletop RPG of gunslingers, haunted doors, dying worlds, psychic songs, and ka-tets bound by fate.
   - CTA: Start Here
   - CTA: Join a Demo Game

2. Short Pitch
   - Explain that players are a ka-tet bound by fate.
   - Explain the Tower, Beams, Red, and broken worlds at a very high level.

3. Four Feature Cards
   - Start Here
   - Player Guide
   - Campaigns
   - Downloads

4. Demo CTA
   - Invite new players to try a one-shot.

5. Visual Strip
   - Door, rose, gun, static, Beam motifs.

### Build Notes

- Use strong art direction.
- Avoid too much lore.
- Keep page scannable.
